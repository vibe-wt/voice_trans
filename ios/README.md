# iOS Workspace

这个目录用于 `AI Voice Journal` 的 iPhone 原生客户端开发。

当前提供：

- `SwiftUI` app 骨架
- `Voice Session` 状态模型
- API / Audio / Calendar / Realtime 服务协议
- `XcodeGen` 配置文件

## 启动方式

如果本机已安装 `XcodeGen`，在本目录执行：

```bash
xcodegen generate
```

然后打开生成的 `AIJournalApp.xcodeproj`。
